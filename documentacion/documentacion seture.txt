1.- crear las carpetas controllers y views, en views copiar las carpetas img, js, css
2.- para que se pueda mandar el correo correctamente se debe descargar "composer require phpmailer/phpmailer"
    -se crea la carpeta extensions y en cmd en esa carpeta se ejecuta el comando composer 
    - se coloca en el index el autoload
    -se abre php.ini y se configura la siguiente linea "sendmail_path = C:\xampp\mailtodisk\mailtodisk.exe"
    -se reinicia el xampp
